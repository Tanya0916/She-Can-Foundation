
# 🌸 She Can Foundation - Landing Page

A simple and responsive single-page website built for the **She Can Foundation** as part of a frontend development internship task.  
The project is created using basic **HTML, CSS, and a little JavaScript**.

---

##  Project Overview

This webpage represents an NGO landing page that highlights the mission of She Can Foundation — empowering women through education, equality, and opportunity.

The design is kept simple, clean, and beginner-friendly while also adding a bit of interactivity and modern UI styling.

---

## ✨ Features

- NGO Name Header
- Attractive Hero Heading
- About Section describing the organization
- Responsive Image (Women Empowerment theme)
- Interactive Button with click message
- Responsive and modern layout
- Hover effects and smooth UI styling
- Simple JavaScript interaction

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (basic DOM interaction)
